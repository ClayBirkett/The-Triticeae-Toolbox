Create A Filter

To create a filter, do the following:

  1. Traits are grouped into categories. Select a category from the box in the area labeled "Category."
  2. A second box will appear in the area labeled "Phenotype." Select a trait from this box.
  3. A dropdown box will appear in the area labeled "Value" as well as information about the selected trait in the lower portion of the table. Select either "equal to" or "between" from the dropdown box.
  4. If you selected "equal to" from the dropdown box, a text box will appear in the area labeled "Value." Type your desired value in this box. If you selected "between" from the dropdown box, two text boxes will appear in the area labeled "Value." Type a value in each box. Each value represents the beginning or the end of a range values.
  5. If you are satisfied with your selections, click the button labeled "Create!"
  6. Repeat steps 1 to 5 to create additional filters.
