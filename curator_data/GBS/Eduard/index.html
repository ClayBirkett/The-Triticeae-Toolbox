Genotype Experiment Description

1. Hard Red Winter Wheat AM panel (it was genotyped with 90K iSelect and you should have this data deposited to database).
2. Cut with PstI and MseI
3. Mapping was done using bowtie1 (3-steps with decreasing stringency: 1 mismatch, 2 mm, 3 mm)
4. SNP calling with GATK pipeline (default routine described on the website).

Filtering: minimum 50 accessions have data and no multi-allele SNPs. Depth of read coverage was not applied since we use multisample variant calling Approach.
The filtering criteria applied: minimum 50 accessions have data, no multi-allele SNPs. In total after this filtering we have ~ 1.26M SNPs in total.
